# Dinamica Lineal (fuerzas constantes)
---
## 📌 Problemas Propuestos
### Problema 1
```{admonition}  Fuerza mínima para mover un cuerpo
:class: important
Un cuerpo descansa sobre una superficie horizontal rugosa, deducir la mínima fuerza para iniciar el movimiento del cuerpo.
```
---
### Problema 2

```{admonition} Máxima aceleración 
:class: important

Un camion remolcador traslada una plataforma que contiene una transformador muy pesado. Si el camion viaja con velocidad crucero sobre una pista plana horizontal, deducir la maxima aceleración de frenado que puede desarrollar el remolcador sin correr el riesgo que el transformador deslice sobre la plataforma. 

```

---
